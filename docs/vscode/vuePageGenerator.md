# Vue 页面生成器

一个 VSCode 扩展，用于快速生成基于 `ol-base-components` 组件的 Vue CRUD 页面模板。无需联调，开箱即用。

## 安装

在 VSCode 扩展商店中搜索"vue-page-generator"并安装

## 使用方法

### 1. 右键生成页面

1. 在 VSCode 文件资源管理器中右键选择一个文件夹
2. 选择 "生成 CRUD 页面" 菜单项
3. 在弹出的配置面板中填写相关信息
4. 点击 "🚀 生成页面" 按钮
